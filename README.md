# OrbitalGuard Capsule

Sistema IoT para monitoramento inteligente de cápsulas espaciais desenvolvido para o desafio Global Solution 2026 da FIAP.

---

## Sobre o Projeto

O OrbitalGuard Capsule é um sistema embarcado baseado em Arduino capaz de monitorar condições internas de uma cápsula espacial simulada em tempo real.

O projeto utiliza sensores para analisar:

- Temperatura;
- Luminosidade;
- Vibração.

As informações são exibidas em um display LCD 16x2 e alertas visuais são acionados através de LEDs.

---

## Objetivo

Desenvolver uma solução IoT inspirada na indústria espacial utilizando conceitos de:

- Sistemas embarcados;
- Sensores;
- Automação;
- Monitoramento em tempo real;
- Internet das Coisas (IoT).

---

## Tecnologias Utilizadas

- Arduino UNO
- Tinkercad
- Linguagem C/C++
- LCD 16x2
- TMP36
- LDR
- SW-200D

---

## Funcionalidades

✅ Monitoramento de temperatura  
✅ Monitoramento de luminosidade  
✅ Detecção de vibração  
✅ Alertas visuais com LEDs  
✅ Exibição em tempo real no LCD  
✅ Simulação completa no Tinkercad
